# @vividcodeai/embedded-gdb-mcp

MCP server for GDB debugging functionality.

## Installation

```bash
npm install -g @vividcodeai/embedded-gdb-mcp
```

## Usage

Configure in your MCP client:

```json
{
  "mcpServers": {
    "embedded-gdb-debugger": {
      "command": "embedded-gdb-mcp"
    }
  }
}
```

Full documentation: [github.com/vividcode-ai/embedded-gdb-mcp](https://github.com/vividcode-ai/embedded-gdb-mcp)
