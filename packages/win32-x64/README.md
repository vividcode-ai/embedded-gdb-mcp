# @vividcodeai/embedded-gdb-mcp-win32-x64

Prebuilt binary for Windows x64. This is a platform dependency of `@vividcodeai/embedded-gdb-mcp`.

**Do not install this package directly.** Install the parent package instead:

```bash
npm install @vividcodeai/embedded-gdb-mcp
```
